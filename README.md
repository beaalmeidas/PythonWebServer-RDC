# Python Web Server
This is an implementation of a web server in python, without the HTTPServer module.

It was done as the final project for the Computer Networks course, Computer Science 2025.2

<br>

## How to run
**1.** Clone the repository:
```
git clone https://github.com/beaalmeidas/PythonWebServer-RDC.git
```

**2.** Create a .env file from the example, and enter your desired host (localhost is recommended), and port number.

**3.** Open a terminal and run:
```
python server.py
```
